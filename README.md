# Overview #
This application is a simplified training and implementation of an emotion-detection model based on the standard FER 2013 dataset, using Haar Cascades.

# Usage # 
The application is set to predict emotions based on input from webcam by default. However, it could be configured to run on videos or separate images.

Simply install all the requirements in requirements.txt and run Run.py

# Features #
Emotion detection in images and videos

Support for multiple emotions

Real-time processing for video streams
